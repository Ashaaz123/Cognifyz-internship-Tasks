# Cognifyz Internship Tasks

This repository contains a collection of C programming projects and technical tasks completed during my internship at **Cognifyz Technologies**. These tasks demonstrate core programming concepts, data structures, algorithm implementation, and file handling capabilities in C.

---

## 🚀 Featured Applications

### 📊 Student Grade Calculator
* **File:** `Student_Grade_Calculator.c`
* **Description:** A robust utility to manage student data, calculate percentages, and automatically assign grades based on conditional logical criteria.
* **Key Skills:** Structures, arrays, conditional formatting, data validation.

### 🗂️ File Copy Utility
* **File:** `File_Copy.c`
* **Description:** A low-level system simulation script designed to read contents from a source file and clone them to a destination file smoothly.
* **Key Skills:** File I/O operations (`fopen`, `fgetc`, `fputc`), error and exception handling.

### 🔢 Array Sorting Suite
* **File:** `Array_sorting.c`
* **Description:** Implements fundamental sorting algorithms to organize numeric datasets in both ascending and descending order.
* **Key Skills:** Pointers, memory layout manipulation, algorithm optimization.
